# 00 — Principle Catalog

A flat lookup of every cognitive principle this skill applies. Use it to name the principle behind a finding precisely, and to jump to the module that elaborates it. Definitions are one-liners; the module holds the rule, the failure mode, and the fix.

| Principle | One-line definition | Module |
| --- | --- | --- |
| **Anchoring Bias** | The first number/option seen sets the reference point for all that follow. | 01 |
| **Decoy Effect** | A deliberately inferior option makes a target option look clearly superior. | 01 |
| **Centre-Stage Effect** | In a row of options, the middle one is favored. | 01, 03 |
| **Serial Position Effect** | First (primacy) and last (recency) items are remembered; the middle is forgotten. | 01, 03, 08 |
| **Default Bias** | Most users never change a pre-selected default. | 01, 02 |
| **Loss Aversion** | Losing something hurts ~2× more than gaining the equivalent. | 01 |
| **Scarcity** | Limited availability raises perceived value — only if genuine. | 01 |
| **Cashless Effect** | Abstracted payment (wallets, one-click) reduces the pain of paying. | 01 |
| **Temptation Bundling** | Pairing a hard step with an immediate reward drives completion. | 01, 05 |
| **Hick's Law** | Decision time grows with the number and complexity of choices. | 02, 06 |
| **Progressive Disclosure** | Reveal information only when the user's action makes it relevant. | 02, 06 |
| **Banner Blindness** | Users ignore anything that looks like an ad/banner. | 02 |
| **Fitts's Law** | Time to hit a target falls with its size and nearness. | 02 |
| **Decision Fatigue** | Decision quality degrades as the number of prior decisions grows. | 02 |
| **Visual Hierarchy** | Size, weight, color, and position rank what the eye sees first. | 03 |
| **Visual Anchors** | One high-contrast element claims the eye's entry point. | 03 |
| **Von Restorff Effect** | The one thing that differs from its peers is remembered. | 03 |
| **Law of Proximity** | Things placed close together are read as related. | 03 |
| **Signifiers** | Visual cues that reveal how an element is used. | 03 |
| **Law of Similarity** | Things that look alike are assumed to behave alike. | 03 |
| **Selective Attention** | People see only what serves their current goal. | 03, 06 |
| **Aesthetic-Usability Effect** | Beautiful interfaces are perceived as more usable and trustworthy. | 04 |
| **Halo Effect** | The first impression colors judgment of everything after. | 04 |
| **Expectations Bias** | Stated expectations shape perceived experience; under-promise, over-deliver. | 04 |
| **Cognitive Dissonance** | A gap between promise and experience creates distrust. | 04 |
| **Feedforward** | Showing the outcome of an action before it's taken. | 04 |
| **Social Proof / Bandwagon** | People do what similar others are seen doing. | 04 |
| **Singularity Effect** | One vivid, named individual outweighs aggregate statistics. | 04 |
| **Feedback Loop** | Every action needs a visible response within ~200ms. | 04, 07 |
| **Curse of Knowledge** | Experts forget what novices don't know; copy must assume zero context. | 04 |
| **Empathy Gap** | We underestimate how stress/fatigue changes behavior. | 04 |
| **Aha Moment** | The action most correlated with retention — reach it fastest. | 05 |
| **Labor Illusion** | Visible effort makes an outcome feel more valuable and trustworthy. | 05, 07 |
| **Chronoception** | Perceived time; uncertainty, not duration, drives abandonment. | 05, 07 |
| **Goal-Gradient Effect** | Motivation rises as the finish line nears; show what remains. | 05 |
| **Commitment & Consistency** | A small early commitment makes later, larger ones more likely. | 05 |
| **Sunk-Cost Effect** | Accumulated investment makes people reluctant to abandon. | 05 |
| **Endowment Effect** | People overvalue what they feel they own. | 05 |
| **Reactance** | People resist a perceived loss of freedom; every gate needs an exit. | 05, 09 |
| **Variable Reward** | Unpredictable rewards sustain engagement — in discovery, never the core flow. | 05 |
| **IKEA Effect** | We value what we helped build; get users creating early. | 05 |
| **Investment Loops** | Small repeated contributions compound into stickiness. | 05 |
| **Internal Triggers** | The emotion that pulls a user back; design the fastest path from it to relief. | 05 |
| **Zeigarnik Effect** | Unfinished tasks stay mentally active; show the gap specifically. | 05 |
| **Spacing Effect** | Learning spread over time sticks; drip feature education. | 05 |
| **Fresh-Start Effect** | Calendar landmarks (Mon, 1st) are natural re-entry points. | 05 |
| **Method of Loci** | Spatial memory; fix each feature's location and never move it. | 05, 08 |
| **Occam's Razor** | The simplest sufficient solution wins. | 06 |
| **Law of Prägnanz** | The eye resolves complexity into the simplest possible form. | 06 |
| **Cognitive Load** | Total mental effort a screen demands; minimize it. | 06 |
| **Miller's Law** | Working memory holds ~5–7 items; chunk beyond that. | 06 |
| **Chunking** | Grouping items into named units eases processing. | 06 |
| **Unit Bias** | A bounded unit feels like the right amount to complete. | 06 |
| **Flow State** | Uninterrupted focus during active creation; protect it above all. | 06 |
| **Tesler's Law** | Complexity is conserved — absorb it into the system, not the user. | 06 |
| **Affect Heuristic** | Emotional tone shapes decisions; design it deliberately. | 07 |
| **Peak-End Rule** | An experience is remembered by its peak and its end. | 07 |
| **Delighters** | Small unexpected moments of personality, dismissible and non-blocking. | 07 |
| **Hyperbolic Discounting** | Near rewards beat distant ones; deliver value immediately. | 07 |
| **Storytelling Effect** | Narrative is remembered and felt more than facts. | 07 |
| **Recognition Over Recall** | Showing options beats forcing recall from memory. | 08 |
| **Picture Superiority Effect** | Images are remembered far better than words. | 08 |
| **Negativity Bias** | Bad experiences weigh more than good; perfect the worst moments. | 08 |
| **Availability Heuristic** | Recent/vivid evidence dominates judgment. | 08 |
| **Weber's Law** | Change is judged proportionally; roll big changes out in increments. | 08 |
| **Streisand Effect** | Hiding/suppressing information increases suspicion; explain restrictions openly. | 09 |
| **Self-Serving Bias** | We blame circumstances, not ourselves; error copy must own the fault. | 09 |
| **Barnum-Forer Effect** | Generic statements feel personal; reference the user's own content honestly. | 09 |
| **Survivorship Bias** | Studying only successes hides why others left; measure failure too. | 09 |
| **False-Consensus Effect** | We assume others think like us; replace assumptions with sourced data. | 09 |
| **Second-Order Effect (Cobra Effect)** | Incentives produce gamed, unintended outcomes; map them before shipping. | 09 |
